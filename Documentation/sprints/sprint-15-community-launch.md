# Sprint 15 — Community Launch

## Goal

Execute the community outreach plan now that the crate is published, documented, and contributor-ready.

## Why Now

The technical foundation is solid. This sprint is about visibility and adoption — getting the project in front of the people who need it.

## Scope

- LinkedIn announcement post with terminal demo
- Reddit posts to r/rust and r/embedded
- Submit to Awesome Embedded Rust list
- Submit to Awesome AUTOSAR list
- Blog post: "ARXML to running Rust SOME/IP services — zero manual code"
- Engage autosar-data and someip_parse maintainers (acknowledge ecosystem)
- Open GitHub Discussions for Q&A

## Out of Scope

- Conference talk submissions (Phase 6)
- Paid promotion

## Exit Criteria

- At least 3 public posts/submissions made
- GitHub Discussions enabled
- First external star, issue, or discussion from someone outside the project
